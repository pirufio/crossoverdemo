SETUP INSTRUCTIONS

1)Prerequisites:

install node js https://nodejs.org/en/

intall ruby (for compass) http://rubyinstaller.org/downloads/ (windows)

2) open command line and cd into src folder

3) Run on command line:

gem install compass

npm install -g grunt-cli bower

npm install

bower install

grunt serve to run website
